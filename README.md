###  Чтобы развернуть проект необходимо:
```sh
$ https://github.com/OlBol/order-form.git
$ cd order-form
$ npm i 
```

### Скрипты package.json:

| Скрипт | Назначение |
| ------ | ------ |
| dev | Запустить development-режим сбокри. |

##### Запустить скрипт:
```sh
$ npm run имя_скрипта
```

##### Либо:
```sh
$ yarn имя_скрипта
```
